# ملخص التحديثات المضافة

## إضافات Firestore (قواعد الأمان):

### 1. المنشورات (posts):
- قراءة: للجميع
- إنشاء: للمستخدمين المسجلين
- تعديل/حذف: لمالك المنشور فقط
- التعليقات: لا يمكن حذفها (delete: if false)
- الإعجابات: إنشاء/قراءة/حذف فقط

### 2. القصص (stories):
- قراءة: للجميع
- إنشاء: للمستخدمين المسجلين
- تعديل: غير مسموح
- حذف: لمالك القصة فقط

### 3. تقارير المنشورات (postReports):
- قراءة: للمستخدمين المسجلين
- إنشاء: للمستخدمين المسجلين
- تعديل/حذف: غير مسموح

## إضافات المنشورات (posts.html):

### 1. البحث:
- بحث في الأصدقاء والمنشورات
- تبويبات: الكل - الأصدقاء - المنشورات
- عرض النتائج مع إمكانية الضغط للمستخدم لعرض بروفايله

### 2. بروفايل المستخدم:
- نافذة منبثقة لعرض بيانات المستخدم
- عرض الصورة والاسم وعدد المنشورات
- زر مراسلة
- زر إضافة/إلغاء صديق
- عرض منشورات المستخدم

### 3. التفاعل مع المنشورات:
- الضغط على اسم المستخدم أو صورته لفتح بروفايله

## إضافات الغرف (chat-room.html):

### التعليقات:
- عرض التعليقات فقط
- لا توجد أزرار للإعجاب بالتعليقات
- لا توجد أزرار للحذف
- لا توجد إمكانية للرد على التعليقات
- يمكن فقط إضافة تعليق جديد
